# Frontend - Analisador Léxico

Frontend minimal con Vite + Vue 3 que permite pegar código Python y recibir tokens desde el backend.

La interfaz también puede ejecutar el análisis sintáctico y mostrar el AST resultante.

Instalación y ejecución:

```bash
cd frontend
npm install
npm run dev
```

El frontend espera que el backend esté en `http://localhost:3000`.
